# Лабиринт сокровищ

**Лабиринт сокровищ** — это консольная текстовая adventure-игра, написанная на чистом Python.  
Игрок исследует древний лабиринт, перемещается между комнатами, собирает предметы, сталкивается со случайными событиями и ловушками, решает загадки и пытается добраться до главного сокровища.

Проект реализован в учебных целях и предназначен для практического закрепления навыков работы с Python и Git.

---

## Возможности игры

- Перемещение между комнатами (полная карта лабиринта)
- Инвентарь и взаимодействие с предметами
- Загадки, влияющие на дальнейший ход игры
- Случайные события и ловушки
- Несколько вариантов развития событий (победа или поражение)
- Полностью процедурный код (без классов и ООП)

---

## Основные команды в игре

- go <direction> — перейти в направлении (north, south, east, west)
- north / south / east / west — короткая команда перемещения
- look — осмотреть текущую комнату
- take <item> — поднять предмет
- use <item> — использовать предмет из инвентаря
- inventory — показать инвентарь
- solve — решить загадку или попытаться открыть сундук
- help — показать список команд
- quit — выйти из игры
